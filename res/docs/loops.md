# Циклы

Weasel разработан как язык для записи алгоритмов в читаемой форме. Циклы здесь выглядят и читаются как псевдокод — без лишнего синтаксического шума.

Есть три принципиально разных вида циклов:
- **`repeat`** — декларативный перебор: диапазон, коллекция, фиксированное количество
- **`run`** — цикл с условием (аналог `while`)
- **`{ } run ()`** — цикл с постусловием (аналог `do-while`)

Заголовок `repeat` всегда заключается в скобки `( )`:

```wes
repeat (i in 0 .. 10) { }
```

---

## repeat — диапазон

Итерация по целочисленному диапазону `[start, end)`. Правая граница **не включается**.

```wes
repeat (i in 0 .. 5)
{
    print(i)     # 0 1 2 3 4
}
```

Тип итератора выводится из границ. Если границы `uint` — итератор `uint`, если `int` — `int`. Оба выражения должны быть одного типа.

```wes
val uint n = data.size
repeat (i in 0 .. n)
{
    print(data[i])
}
```

Границы могут быть произвольными выражениями:

```wes
repeat (i in lo .. hi)
{
    process(i)
}
```

---

## repeat — диапазон с шагом

Добавьте `step`, чтобы задать произвольный шаг. Шаг должен быть того же типа, что и итератор.

```wes
repeat (i in 0 .. 10 step 2)
{
    print(i)     # 0 2 4 6 8
}
```

```wes
repeat (i in 0 .. n step k)
{
    print(i)
}
```

Шаг всегда положителен — для убывающего диапазона используйте `down`.

---

## repeat — убывающий диапазон

Оператор `down` задаёт итерацию от большего к меньшему. Синтаксис `i in start down end` — итератор убывает от `start` до `end` (не включая `end`).

```wes
repeat (i in 5 down 0)
{
    print(i)     # 5 4 3 2 1
}
```

Шаг убывания всегда равен 1. Если нужен другой шаг — используйте `run`.

---

## repeat — перебор коллекции

Итерация по всем элементам массива. Weasel сам управляет индексом: вы получаете только значение.

```wes
val array[int] nums = [1, 2, 3, 4, 5]

repeat (n in nums)
{
    print(n)     # 1 2 3 4 5
}
```

Тип переменной-итератора выводится из типа элементов массива автоматически.

---

## repeat — перебор с индексом

Если нужен и индекс, и значение — объявите оба через запятую. Первый — индекс (тип `int`), второй — значение.

```wes
repeat (i, n in nums)
{
    print(i, ": ", n)
}
```

```wes
val array[string] names = ["Алиса", "Боб", "Чарли"]

repeat (idx, name in names)
{
    println(idx, " -> ", name)
    # 0 -> Алиса
    # 1 -> Боб
    # 2 -> Чарли
}
```

---

## repeat — фиксированное количество

Когда нужно просто повторить тело N раз и не нужен счётчик.

```wes
repeat (5 times)
{
    println("привет")
}
```

Число итераций может быть переменной:

```wes
val int n = 10
repeat (n times)
{
    doStep()
}
```

> **Ограничение:** в `times` сейчас поддерживаются только имя переменной или числовой литерал, не произвольное выражение. Для сложного выражения вычислите его заранее: `val int count = a * b + c`.

---

## run — цикл с предусловием

Выполняет тело, пока условие истинно. Если с самого начала условие `false` — тело не выполнится ни разу.

```wes
var int i = 0
run (i < 5)
{
    print(i)
    i = i + 1
}
```

Условие обязано быть типа `bool`. Weasel не имеет понятия "truthy" — нельзя писать `run (count)` или `run (ptr)`.

---

## do-run — цикл с постусловием

Тело выполняется **хотя бы один раз**: сначала тело, потом проверка условия.

```wes
var int x = 100

{
    println(x)
    x = x - 1
} run (x > 0)
```

Используйте `do-run` когда первый шаг всегда нужен: чтение ввода, инициализация, первый шаг алгоритма.

---

## break и continue

`break` немедленно завершает текущий цикл. `continue` пропускает остаток тела и переходит к следующей итерации.

```wes
repeat (i in 0 .. 100)
{
    when
    {
        i == 7 -> break
    }
    println(i)
}
# Выведет 0 1 2 3 4 5 6
```

```wes
var int sum = 0
repeat (i in 0 .. 10)
{
    when
    {
        i % 2 == 0 -> continue
    }
    sum = sum + i
}
println(sum)    # 1+3+5+7+9 = 25
```

`break` и `continue` всегда относятся к **ближайшему** объемлющему циклу. Внешние циклы не затрагиваются.

Оба оператора — самостоятельные инструкции, пишутся на отдельной строке. Компилятор проверяет, что они используются только внутри тела цикла.

---

## Вложенные циклы

Любые виды циклов можно свободно вкладывать. Итераторы разных уровней не конфликтуют.

```wes
repeat (i in 1 .. 4)
{
    repeat (j in 1 .. 4)
    {
        print(i * j, "\t")
    }
    println("")
}
```

---

## Выбор подходящего цикла

| Ситуация | Цикл |
|---|---|
| Перебор диапазона `0..n` | `repeat (i in 0 .. n)` |
| Диапазон с нестандартным шагом | `repeat (i in 0 .. n step k)` |
| Обратный порядок | `repeat (i in n down 0)` |
| Перебор всех элементов массива | `repeat (v in arr)` |
| Перебор с индексом | `repeat (i, v in arr)` |
| Повторить ровно N раз | `repeat (n times)` |
| Условие неизвестно заранее | `run (cond)` |
| Первый шаг всегда нужен | `{ } run (cond)` |
