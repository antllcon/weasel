# Минимальная структура программы

.const — Пул статических констант (числа). Строки пока предполагаются там же или отдельной сущностью. Индексация начинается с 0.
.code — Секция исполняемого кода, где располагаются инструкции и метки.

```wesbc
# Секция пула констант
.const
42          # cst 0: Просто число 42

# Секция байткода
.code
_start:
    const 0 # Загрузить константу по индексу 0 на стек
    ret     # Вернуть верхнее значение стека и завершить выполнение
```

### Справочник инструкций

Аргументы: указываются сразу после мнемоники (например, const <idx>).
Стек: показывает состояние вершины стека ДО -> ПОСЛЕ выполнения. Вершина стека всегда находится справа.
