# Ветвление: if / else

Иногда программа должна принимать решения: сделать одно при выполнении условия и другое — если условие не выполнено. Для таких ситуаций в Weasel есть конструкция `if`.

Как и во всём языке, условие обязано иметь тип `bool`. В Weasel нет понятия "truthy" — число `0` не равно `false`, а непустая строка не равна `true`. Это устраняет целый класс неочевидных ошибок.

---

## Простое ветвление

```wes
val int temperature = -5

if (temperature < 0)
{
    println("Мороз!")
}
```

Если условие ложно, блок пропускается целиком.

---

## Две ветви: if / else

```wes
val int score = 72

if (score >= 60)
{
    println("Зачёт")
}
else
{
    println("Незачёт")
}
```

Ровно одна из двух ветвей выполнится при любом значении.

---

## Цепочка условий: else if

Когда вариантов больше двух, условия выстраиваются в цепочку. Проверка идёт сверху вниз, и выполняется **первая** подошедшая ветка — остальные игнорируются.

```wes
val int grade = 85

if (grade >= 90)
{
    println("Отлично")
}
else if (grade >= 75)
{
    println("Хорошо")
}
else if (grade >= 60)
{
    println("Удовлетворительно")
}
else
{
    println("Неудовлетворительно")
}
```

Финальный `else` необязателен. Если ни одно условие не выполнилось и `else` нет, выполнение просто продолжается дальше.

---

## Вложенные if

`if` можно вкладывать друг в друга на любую глубину. Фигурные скобки делают структуру очевидной.

```wes
val int x = 3
val int y = 7

if (x > 0)
{
    if (y > 5)
    {
        println("x положительный, y большой")
    }
    else
    {
        println("x положительный, y маленький")
    }
}
else
{
    println("x не положительный")
}
```

`else` всегда принадлежит ближайшему `if` — фигурные скобки делают это правило явным и не оставляют места для двусмысленности.

---

## Условия

В качестве условия можно использовать любое выражение типа `bool` — сравнение, логическую операцию или их комбинацию.

```wes
val int a = 3
val int b = 7

# Логическое "И": обе части должны быть истинны
if (a > 0 and b > 0)
{
    println("Оба положительные")
}

# Логическое "ИЛИ" с коротким замыканием
if (a > 10 or b > 5)
{
    println("Хотя бы одно больше порога")
}

# Отрицание
if (not (a == b))
{
    println("a и b различаются")
}
```

`and` и `or` вычисляются лениво: если левая часть `and` ложна, правая не вычисляется вовсе; если левая часть `or` истинна — правая тоже.

## if с return

`return` внутри `if` немедленно завершает функцию. Это позволяет писать функции в стиле "ранних выходов" — без глубокой вложенности.

```wes
int <- sign(int x)
{
    if (x > 0)
    {
        return 1
    }
    else if (x < 0)
    {
        return -1
    }
    return 0
}
```

---

## if внутри цикла

Внутри `if` можно использовать `break` и `continue` — они управляют объемлющим циклом.

```wes
repeat (i in 0 .. 10)
{
    if (i == 7)
    {
        break           # выходим из цикла при i == 7
    }
    if (i % 2 == 0)
    {
        continue        # пропускаем чётные
    }
    println(i)          # выведет: 1 3 5
}
```
