# Базовые типы

В Weasel строгая статическая типизация. Это означает, что тип каждой переменной известен компилятору заранее, что исключает множество случайных ошибок во время выполнения. В языке нет скрытого (неявного) приведения типов  — любые преобразования должны быть выражены явно.

Несмотря на наличие фиксированного набора базовых типов вы можете создавать собственные структуры данных (struct, union, enum), которые будут работать так же органично и эффективно, как и встроенные типы.

### Числа

Для представления чисел Weasel предоставляет три фундаментальных типа. Чтобы вы могли не беспокоиться о внезапных переполнениях в повседневных задачах, все числа под используют 64-битное представление памяти.

#### Целые числа

- `int`. Знаковое целое число. Покрывает огромный диапазон от отрицательных до положительных значений. Это тип по умолчанию для большинства математических вычислений. (сказа)

- `uint`. Беззнаковое (натуральное) целое число. Идеально семантически подходит для описания размеров, количеств, счетчиков и индексов.

#### Числа с плавающей точкой

- `real`. Число с плавающей точкой двойной точности по стандарту (IEEE 754). Используется там где требуется высокая точность дробных значений.

``` wes
val int temperature = -15
val uint score = 1050
val real pi = 3.14159265ф
```

### Логический тип

Тип `bool` представляет логические значения. Он используется в ветвлениях (`when`, `if`) и циклах (`run`, `rep`).
У типа bool есть только два встроенных значения: `true` и `false`.

``` wes
val bool isCompilationSuccessful = true
val bool hasErrors = false
```

### Строки

Для работы с текстовыми данными используется тип `string`. Строковые литералы всегда заключаются в двойные кавычки (" "). Управление строками выполняется через динамическую память.

``` wes
val string greeting = "Welcome to Weasel Language!"
```

### Пустой тип

Особый тип `void` указывает на отсутствие осмысленного значения. В первую очередь он используется для обозначения того, что функция выполняет какое-то действие (например, выводит текст на экран), но ничего не возвращает вызывающему коду. Создать переменную типа `void` нельзя.

``` wes
void <- printGreeting() 
{
    print("Hello!")
}
```

### Приведение типов

В Weasel безопасность превыше всего: компилятор не станет тайно преобразовывать `int` в `real` при сложении. Все конвертации типов осуществляются программистом осознанно.

Для приведения одного скалярного типа к другому язык использует универсальный синтаксис вызова методов. Это делает код читаемым слева направо.

``` wes
val int negativeScore = -120
val uint itemsCount = 250
val real preciseMath = 2.71828

# 1. Извлечение целой части из дроби
val int truncated = preciseMath.int()   # Результат: 2

# 2. Превращение целого в дробное
val real asFloat  = itemsCount.real()   # Результат: 250.0

# 3. Приведение между знаковыми и беззнаковыми
val uint safeCast = negativeScore.uint()
```

### Расширяемость

Базовые типы — это лишь фундамент. Благодаря заложенной поддержке структурного программирования, язык будет позволять вам конструировать собственные типы любой сложности:

``` wes
struct Vector3 
[
    val real x
    val real y
    val real z
]
```

Такие пользовательские типы поддерживают инициализацию, перемещение в памяти (через оператор <- ) и сборку мусора наравне с базовыми `string` или `int`.
