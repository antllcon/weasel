# Модули и импорт

Программу можно разбить на несколько файлов. Каждый `.wes`-файл — это **модуль**: набор объявлений (функций, классов, перечислений, глобальных констант). Один модуль подключает другой директивой `import`.

---

## Импорт

`import` указывает имя модуля строкой и пишется на уровне файла:

```wes
import "util"
import "math/vec"

void <- main()
{
    println(square(7))
}
```

Расширение `.wes` дописывать не нужно. Путь может содержать подкаталоги (`"math/vec"` → `math/vec.wes`).

Импортируемый модуль **не обязан** содержать `main` — это библиотека. Точка входа `main` нужна только главному файлу программы.

```wes
# util.wes — библиотека без main
int <- square(int x)
{
    return x * x
}
```

---

## Как разрешается путь

Для каждого `import "имя"` модуль ищется по порядку:

1. рядом с импортирующим файлом (`<каталог файла>/имя.wes`);
2. в каталоге стандартной библиотеки (`<stdlib>/имя.wes`).

Берётся первый найденный. Каталог stdlib по умолчанию — `stdlib`; его можно переопределить флагом `--stdlib <каталог>`.

Если модуль не найден — ошибка компиляции с указанием файла и строки импорта.

---

## Единое пространство имён

Объявления всех импортированных модулей **сливаются в одно пространство имён** ещё до анализа программы. После импорта имена из модуля доступны напрямую — без префиксов:

```wes
import "util"

void <- main()
{
    var Greeter g = Greeter("Weasel")   # класс из util
    println(g.greeting())
    println(square(7))                  # функция из util
}
```

Поскольку имена общие, **два модуля не могут объявить одну и ту же сущность** с одинаковой сигнатурой — это даст ошибку «повторное объявление». Порядок импортов и объявлений значения не имеет.

---

## Повторные и циклические импорты

Каждый модуль загружается **ровно один раз**. Если несколько файлов импортируют один и тот же модуль, он подключается единожды. Циклические импорты (`a` → `b` → `a`) безопасны и не приводят к зацикливанию — повторный заход в уже загруженный модуль игнорируется.

---

## Пример

```
res/tests/modules/
├── util.wes     # square(), класс Greeter — без main
└── app.wes      # import "util", main()
```

```wes
# app.wes
import "util"

void <- main()
{
    println(square(7))         # 49
    var Greeter g = Greeter("Weasel")
    println(g.greeting())      # Weasel
}
```

Запуск (из каталога `res`):

```
wesc tests/modules/app.wes
```
